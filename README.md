# SalesOrder

A very simple web application to manages sales order.

### Technologies:
Target Framework: .NET 5

IDE: Visual Studio 2019

#### UI
Blazor Webassembly 5.0.17 with Syncfusion library (UI Data Grid) 19.4.0.56

#### Server
.NET 5 WEB API

#### Database
SQLite
